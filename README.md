# Forecast App

## How to run

1- Clone the repository
2- Cd to the project folder
3- Execute

  ### Using Docker
    docker-compose up -d --build
   The app will open up on `localhost:3001`
  
  ### Using yarn
    yarn execute
   The app will open up on `localhost:3000`
   
   ### Using npm
    npm run execute
   The app will open up on `localhost:3000`
